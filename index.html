Sincronización con GitHub

Guarda los datos de todos los usuarios en un repositorio de GitHub para acceder desde cualquier dispositivo.

Crea un token de acceso fine-grained en GitHub limitado a este repositorio, con permiso de lectura y escritura sobre "Contents". El token se guarda únicamente en este navegador/dispositivo. Usa un repositorio privado: los datos (historiales, medidas, fotos, vídeos, contraseñas cifradas) quedarán dentro del archivo JSON del repositorio.
