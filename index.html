Sincronización con GitHub

Guarda los datos de todos los usuarios en un repositorio de GitHub para acceder desde cualquier dispositivo.

Crea un token de acceso fine-grained en GitHub limitado a este repositorio, con permiso de lectura y escritura sobre "Contents". El token se guarda únicamente en este navegador/dispositivo. Usa un repositorio privado. Los datos se guardan en varios archivos dentro de la carpeta indicada: accounts.json, media.json, ranking.json, exercises.json y social.json.
